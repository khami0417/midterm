DREAMCATCHER

Dreamcatcher是Happy Face娛樂在2017年推出的七人女子團體。其前身為2014年出道的五人女子團隊MINX，原本成員包括祗攸、始娟、秀雅、多美、裕賢，於2017年改成Dreamcatcher時加入新成員HanDong和GaHyeon。官方粉絲名稱為「인썸니아(InSomnia)」。 In是英文在...之中的意思，而Somnia是拉丁語夢中的意思，意謂著在夢裡面Dreamcatcher跟粉絲們見面的幸福的時光，永遠在一起。
